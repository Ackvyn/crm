# Security Policy

## Supported versions

Ackvyn CRM is maintained on the latest `main` / published CDN + Worker template.
Older snapshots are not separately patched unless a critical issue affects many
operators.

## Reporting a vulnerability

Please email **hello@ackvyn.org** with:

- A short description of the issue
- Impact (e.g. auth bypass, data exposure, Worker DoS)
- Steps or a proof of concept if you have one
- Whether the report may be disclosed after a fix

Do not open a public GitHub issue for security bugs.

We will respond as soon as practical. There is no bug bounty program today; good-faith
reports may be acknowledged in release notes when appropriate.
