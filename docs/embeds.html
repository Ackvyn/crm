Embeds

Scripts are loaded from the CDN (this site). They talk to the operator’s Worker via data-crm / data-api.

Floating chat

<script
  src="https://crm.ackvyn.org/cdn/embed.js"
  data-site="your-site-key"
  data-crm="https://YOUR-WORKER.workers.dev"
  data-mode="float"
  async
></script>

Inline mount

<div id="ackvyn-crm-chat"></div>
<script
  src="https://crm.ackvyn.org/cdn/embed.js"
  data-site="your-site-key"
  data-crm="https://YOUR-WORKER.workers.dev"
  data-mode="inline"
  data-target="#ackvyn-crm-chat"
  async
></script>

Intake

POST https://YOUR-WORKER.workers.dev/v1/{site}/intake
Content-Type: application/json
