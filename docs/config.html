Config reference

Script attributes

  • data-site — site key on that operator’s hub
  • data-crm / data-api — that operator’s Worker origin (required when scripts load from the CDN)
  • data-mode — float | inline
  • data-target — inline mount selector

Example operator config

{
  "scriptCdn": "https://crm.ackvyn.org/cdn",
  "crmApi": "https://my-crm.myaccount.workers.dev",
  "adminUi": "https://crm.ackvyn.org/app/",
  "gitRepo": "myorg/my-pages-repo",
  "gitPathPrefix": "crm-data"
}
