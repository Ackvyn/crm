Google reviews

Configure Place ID in your CRM console. Your Worker syncs to public JSON under your git store: 

crm-data/{site}/reviews.json

Or read the Worker cache:

GET https://YOUR-WORKER.workers.dev/v1/{site}/reviews

Loader (CDN script → your Worker)

<script
  src="https://crm.ackvyn.org/cdn/reviews-embed.js"
  data-site="your-site-key"
  data-crm="https://YOUR-WORKER.workers.dev"
  async
></script>
<script>
  window.addEventListener('ackvyn-reviews', (e) => {
    // build your own UI from e.detail
  })
</script>
